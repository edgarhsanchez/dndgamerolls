# Dice FX textures

Drop textures into the folders below using the exact filenames.
These files are embedded into the executable via `Dice3dEmbeddedAssetsPlugin`.

## Fire (d20 rolls 20)
- `assets/fx/fire/noise.png` (tileable noise)
- `assets/fx/fire/ramp.png` (1D/2D color ramp)
- `assets/fx/fire/mask.png` (tileable mask)

## Atomic (two+ d20 roll 20)
- `assets/fx/atomic/noise.png` (tileable noise)
- `assets/fx/atomic/ramp.png` (1D/2D color ramp)
- `assets/fx/atomic/mask.png` (tileable mask)

## Electric (any die rolls max)
- `assets/fx/electric/noise.png` (tileable noise)
- `assets/fx/electric/ramp.png` (1D/2D color ramp)
- `assets/fx/electric/mask.png` (tileable crack/lightning mask)

Notes:
- You can replace the placeholder PNGs with your own images.
- Keep filenames the same so embedding keeps working.
