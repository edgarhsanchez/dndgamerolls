# Example: Multiple Dice with Skill Check

## Command
```bash
dice3d --dice 1d20 --dice 2d6 --checkon stealth
```

## Description
Rolls 1d20 + 2d6 with the character's stealth modifier applied.

## Expected Result
- 3 dice spawn: 1 yellow D20, 2 black D6
- Results show individual dice values
- Modifier (+9 for Stealth with Expertise) is applied to total
- Display shows: "Modifier: stealth (+9)"

## In-App Alternative
Press `/` and type:
```
--dice 1d20 --dice 2d6 --checkon stealth
```
