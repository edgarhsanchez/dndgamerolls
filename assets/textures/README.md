# Dice Textures

Place dice textures here for custom appearance.

## Expected Files

- `d4_texture.png`
- `d6_texture.png`
- `d8_texture.png`
- `d10_texture.png`
- `d12_texture.png`
- `d20_texture.png`

## Texture Format

- **Format**: PNG (with alpha) or JPEG
- **Size**: 512x512 or 1024x1024 recommended
- **Type**: Diffuse/Albedo texture

## PBR Textures (Optional)

For physically-based rendering, you can also include:
- `*_normal.png` - Normal map
- `*_roughness.png` - Roughness map  
- `*_metallic.png` - Metallic map
