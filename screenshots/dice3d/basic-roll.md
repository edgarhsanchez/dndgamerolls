# Example: Basic 1d20 Roll

## Command
```bash
dice3d
```

## Description
Launches the 3D dice simulator with a single D20 (default).

## Controls
- **SPACE** - Re-roll the dice
- **R** - Clear all dice
- **/** or **Enter** - Open command input
- **1-9** - Reroll from command history

## Expected Result
- Crystal-yellow D20 spawns in glass dice box
- Dice rolls with physics simulation
- Result appears in top-left when dice settles
- Command history panel visible on right side
