# Microsoft Store Assets

This directory should contain the following image assets for Microsoft Store submission:

## Required Assets

| File | Size | Description |
|------|------|-------------|
| `StoreLogo.png` | 50x50 | Store logo |
| `Square44x44Logo.png` | 44x44 | Small tile |
| `Square150x150Logo.png` | 150x150 | Medium tile |
| `Square310x310Logo.png` | 310x310 | Large tile |
| `Wide310x150Logo.png` | 310x150 | Wide tile |
| `SplashScreen.png` | 620x300 | Splash screen |

## Design Guidelines

- Use transparent backgrounds where appropriate
- Primary color: `#1a1a2e` (dark purple/blue)
- Accent color: `#e94560` (red)
- Include dice imagery (D20 recommended for main logo)

## Tools

You can create these assets using:
- [Figma](https://www.figma.com/)
- [Canva](https://www.canva.com/)
- [GIMP](https://www.gimp.org/)
